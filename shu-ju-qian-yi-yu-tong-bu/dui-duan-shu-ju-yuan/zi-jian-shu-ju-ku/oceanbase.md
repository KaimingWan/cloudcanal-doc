# OceanBase

### 核心能力支持矩阵

| 功能       | 是否支持 | 备注          |
| -------- | ---- | ----------- |
| 结构迁移     | 是    |             |
| 全量数据迁移   | 是    |             |
| 定时全量     | 否    |             |
| 增量实时同步   | 是    |             |
| 数据校验     | 是    |             |
| 定时校验     | 是    |             |
| 数据订正     | 是    |             |
| 位点管理     |      | 支持位点回溯、位点重置 |
| 编辑订阅     | 是    |             |
| DDL实时同步  | 是    |             |
| where件过滤 | 是    |             |
| 对象映射     | 是    | 支持库、表、列的映射  |
| 自定义代码    | 是    |             |
| 整库迁移     | 否    |             |
